Aumenta o limite de memória do WordPress para 1024MB para melhorar o desempenho do WooCommerce.

== Description ==

Este plugin aumenta o limite de memória do WordPress para 1024MB, o que melhora significativamente o desempenho do WooCommerce, especialmente em sites com muitos produtos ou com plugins que consomem muita memória.

Características:
* Aumenta o limite de memória do PHP para 1024MB
* Define as constantes WP_MEMORY_LIMIT e WP_MAX_MEMORY_LIMIT
* Adiciona uma página de administração para verificar as configurações de memória
* Exibe informações de uso de memória no rodapé do painel de administração

== Installation ==

1. Faça o upload do plugin para o diretório `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Acesse a página de configurações em 'Ferramentas > Configurações de Memória'

== Frequently Asked Questions ==

= Por que aumentar o limite de memória? =

O WordPress e o WooCommerce podem consumir muita memória, especialmente em sites com muitos produtos ou com plugins adicionais. Aumentar o limite de memória pode resolver problemas de desempenho e erros de "memória esgotada".

= Como sei se o plugin está funcionando? =

Acesse a página 'Ferramentas > Configurações de Memória' para verificar as configurações atuais de memória.
