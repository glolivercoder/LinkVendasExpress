Template personalizado para o WooCommerce que mantém o mesmo layout do app PDV Vendas.

== Description ==

O PDV Vendas Theme é um plugin que personaliza o tema do WooCommerce para manter o mesmo layout do aplicativo PDV Vendas. Ele adiciona estilos CSS personalizados e funcionalidades específicas para integração com o sistema PDV Vendas.

Principais recursos:

* Layout semelhante ao aplicativo PDV Vendas
* Suporte para produtos do PDV Vendas
* Exibição de informações do PDV Vendas na página do produto
* Estilos CSS personalizados para uma experiência de usuário consistente

== Installation ==

1. Faça upload do arquivo ZIP do plugin através do painel de administração do WordPress (Plugins > Adicionar Novo > Enviar Plugin)
2. Ative o plugin
3. O tema será aplicado automaticamente ao seu site WooCommerce

== Frequently Asked Questions ==

= O plugin é compatível com qualquer tema do WordPress? =

Sim, o plugin foi projetado para funcionar com qualquer tema do WordPress que tenha suporte para WooCommerce.

= Como posso personalizar as cores do tema? =

As cores do tema podem ser personalizadas editando o arquivo CSS do plugin ou usando o Personalizador do WordPress.
