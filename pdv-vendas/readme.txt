Integração entre o PDV Vendas e o WordPress/WooCommerce.

== Description ==

O plugin PDV Vendas Integration permite a sincronização bidirecional entre o sistema PDV Vendas e o WordPress/WooCommerce.

Principais recursos:

* Sincronização de produtos do PDV Vendas para o WordPress
* Atualização automática de estoque
* Webhook para notificações de vendas
* Suporte a autenticação JWT
* API REST para integração

== Installation ==

1. Faça upload do plugin para o diretório `/wp-content/plugins/`
2. Ative o plugin através do menu 'Plugins' no WordPress
3. Configure as credenciais de API no arquivo do plugin
4. Certifique-se de que o plugin JWT Authentication está instalado e configurado

== Frequently Asked Questions ==

= É necessário o WooCommerce para usar este plugin? =

Sim, o plugin depende do WooCommerce para funcionar corretamente.

= Como configurar o webhook? =

O webhook é configurado automaticamente quando você usa a função "Configurar Webhook" no PDV Vendas.
