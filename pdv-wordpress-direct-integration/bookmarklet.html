PDV Vendas - WordPress Sync Bookmarklet

Este bookmarklet permite sincronizar facilmente os elementos selecionados no PDV Vendas com o WordPress.

Instruções

  1. Arraste o botão abaixo para a barra de favoritos do seu navegador
  2. Acesse o PDV Vendas
  3. Inicie o MCP Browser Tools com o comando: npx @agentdeskai/browser-tools-mcp@latest
  4. Selecione um elemento usando o MCP Browser Tools
  5. Clique no bookmarklet na sua barra de favoritos
  6. O elemento selecionado será sincronizado com o WordPress

Arraste este botão para sua barra de favoritos:

PDV → WordPress Sync

Nota

Se o bookmarklet não funcionar, você pode copiar e colar o seguinte código diretamente no console do navegador:

fetch('https://achadinhoshopp.com.br/loja/wp-content/plugins/pdv-wordpress-direct-integration/assets/js/pdv-sync-bookmarklet.js')
.then(response => response.text())
.then(code => {
    const script = document.createElement('script');
    script.textContent = code;
    document.body.appendChild(script);
});
